# Estudos ADS

Repositório criado para organizar minhas anotações, atividades e exercícios da graduação em Análise e Desenvolvimento de Sistemas pela UNIASSELVI.

Este espaço será usado como um caderno público de estudos, registrando minha evolução em programação, banco de dados, infraestrutura, redes, Linux, Git/GitHub e desenvolvimento de sistemas.

## Objetivo

Organizar meus estudos da faculdade e dos cursos complementares, mantendo um histórico da minha evolução na área de tecnologia.

## Áreas de estudo

- Lógica de programação
- Arquitetura de computadores
- Banco de dados
- HTML, CSS e JavaScript
- Git e GitHub
- Linux e redes
- Desenvolvimento de sistemas
- Infraestrutura de TI
