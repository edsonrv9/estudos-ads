# Lógica e Técnicas de Programação

Esta pasta reúne anotações, atividades e materiais da disciplina Lógica e Técnicas de Programação, cursada no Módulo 1 do curso de Análise e Desenvolvimento de Sistemas pela UNIASSELVI.

## Informações da disciplina

- Disciplina: Lógica e Técnicas de Programação
- Código: ADS05
- Módulo: 1
- Período: 2026/1
- Status: Concluída

## Objetivo da disciplina

Registrar os principais aprendizados sobre lógica de programação, construção de algoritmos, resolução de problemas computacionais e fundamentos iniciais para o desenvolvimento de sistemas.

## Conteúdos relacionados

- Algoritmos
- Variáveis
- Tipos de dados
- Operadores
- Estruturas condicionais
- Estruturas de repetição
- Entrada e saída de dados
- Fluxogramas
- Pseudocódigo
- Resolução de problemas

## Status

Disciplina concluída no Módulo 1.
