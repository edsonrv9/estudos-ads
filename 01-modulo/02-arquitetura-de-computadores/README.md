# Arquitetura de Computadores

Esta pasta reúne anotações, atividades e materiais da disciplina Arquitetura de Computadores, cursada no Módulo 1 do curso de Análise e Desenvolvimento de Sistemas pela UNIASSELVI.

## Informações da disciplina

- Disciplina: Arquitetura de Computadores
- Código: INF14
- Módulo: 1
- Período: 2026/1
- Status: Concluída

## Objetivo da disciplina

Registrar os principais aprendizados sobre os componentes de um computador, funcionamento do hardware, processamento de dados, memória, armazenamento e organização básica dos sistemas computacionais.

## Conteúdos relacionados

- Componentes de hardware
- Processador
- Memória RAM
- Armazenamento
- Placa-mãe
- Barramentos
- Sistemas de entrada e saída
- Organização e funcionamento dos computadores
- Relação entre hardware e software

Disciplina concluída no Módulo 1.
